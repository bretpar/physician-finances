# Project Memory

Fintech design: light theme, Inter font, blue primary (#0066CC), teal accent.
Mock data in src/lib/mockData.ts — no backend connected yet.

- [Withholding engine](mem://features/withholding-engine) — Global withholding method in Settings, per-entry recommendations
- [Income recommendation](mem://features/income-recommendation) — Modal 2 recommendation flow after saving income
- [Projected matching](mem://features/projected-matching) — Matching projected streams to actual income entries
- [K-1 owner deductions](mem://features/k1-owner-deductions) — Owner healthcare/retirement/pre-tax separate from business expenses
